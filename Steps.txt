📝 คู่มือการติดตั้งโปรเจกต์ Rubber Bot สำหรับทีม (Setup Guide)

1. ติดตั้งและรันฝั่ง Python (AI Service) เปิด Terminal จอที่ 1:
-----------------------------------------------------
Bash
-----------------------------------------------------
cd ai-service
pip install -r requirements.txt
python app.py
-----------------------------------------------------
2. ติดตั้งและรันฝั่ง Node.js (Bot Server) เปิด Terminal จอที่ 2:
-----------------------------------------------------
Bash
-----------------------------------------------------
cd bot-server
npm install
node index.js
-----------------------------------------------------
3. เปิด Ngrok (เพื่อให้ LINE เห็นเครื่องเรา) เปิด Terminal จอที่ 3:
-----------------------------------------------------
Bash
-----------------------------------------------------
ngrok http 3000
(แล้วเอา URL https ไปใส่ใน LINE Webhook)
